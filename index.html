Doctasia is a straightforward static website generator for Markdown documentation.

Features

  • multi pages
  • customizable style
  • work out of the box

Quick start

  $> npm install -g doctasia
  $> doctasia

Configuration

You can configure how Doctasia generates your documentation with a manifest file. By default doctasia looks for a file named documentation.json in the cwd.
